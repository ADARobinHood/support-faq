
Les étapes décrites ci-dessous partent du principe que vous avez déjà [installé Yoroi](Wallets/Yoroi/yoroi.md#install)

!> Ne suivez pas ces instructions si vous utilisez un Hardware Wallet

##### Créer un Wallet sur l'extension Yoroi

Sélectionnez `Create Wallet` sur la page d'accueil de Yoroi ci-dessous :

![Home Page](https://raw.githubusercontent.com/cardano-community/support-faq/images/docs/images/yoroi-home.jpg ':size=50%')

Sélectionnez la devise `Cardano` :

![Currency](https://raw.githubusercontent.com/cardano-community/support-faq/images/docs/images/yoroi-select-currency.jpg ':size=30%')

Sélectionnez le type de wallet (pour cette section - puisqu'il s'agit de Hot wallet, nous utiliserons `Create Wallet`)

![Create Wallet](https://raw.githubusercontent.com/cardano-community/support-faq/images/docs/images/yoroi-create-wallet.jpg ':size=30%')

Entrez un nom de wallet pour l'identifier. Vous devrez également saisir un mot de passe de 10 caractères (minimum). Ce mot de passe fera office de protection de deuxième niveau et sera utilisé pour sécuriser (par cryptage) vos clés dans la mémoire cache. *Pour effectuer des virements sortants, vous aurez besoin de ce mot de passe de dépense* Continuez en cliquant sur `Create Personal Wallet`.

![Create Wallet](https://raw.githubusercontent.com/cardano-community/support-faq/images/docs/images/yoroi-new-wallet.jpg ':size=30%')

Vous arriverez à un message de sécurité vous demandant de **confirmer** que personne ne regarde votre écran, car la prochaine étape affichera la clé racine de votre portefeuille - qui dérive vos [clés privées](wallets.md#basics-about-private-keys). Munissez-vous d'un papier et d'un stylo pour écrire la suite de mots qui vous est donnée avant de continuer.

![Accept Warning for exposing keys](https://raw.githubusercontent.com/cardano-community/support-faq/images/docs/images/yoroi-seed-acceptance.jpg ':size=30%')

La page vous demandera d'écrire maintenant la phrase de récupération de 15 mots affichée à l'écran. Veuillez noter soigneusement et vérifier les mots dans l'ordre correct avant de continuer. Souvent, les utilisateurs ont tendance à mélanger l'ordre ou à renommer quelques mots pour des raisons de sécurité, mais oublient de se souvenir de ce qui a été fait.
**Important** : Ces mots sont le **seul** moyen pour vous d'avoir accès à vos fonds. Vous pouvez perdre votre ordinateur et avoir accès aux fonds si vous avez ces mots, mais si vous perdez ces mots - **personne ne peut récupérer les fonds pour vous**.

![Write Down your Recovery Phrase](https://raw.githubusercontent.com/cardano-community/support-faq/images/docs/images/yoroi-seed.jpg ':size=30%')

Maintenant, il vous sera demandé de vérifier la phrase de récupération que vous avez notée. Veuillez sélectionner les mots de la liste dans le même ordre que celui dans lequel vous les avez vus à l'écran précédent.

Une fois que vous avez saisi les 15 mots, un rappel sur l'importance de ces mots secrets s'affiche. Veuillez *lire* et ensuite cocher les cases pour confirmer :

![Confirmation about Recovery Phrase](https://raw.githubusercontent.com/cardano-community/support-faq/images/docs/images/yoroi-confirm-seed.jpg ':size=30%')

Cela devrait vous amener à l'écran suivant.

![Wallet Created Successfully](https://raw.githubusercontent.com/cardano-community/support-faq/images/docs/images/yoroi-wallet-created.jpg ':size=30%')

Félicitations ! Vous avez créé votre portefeuille Yoroi.

##### Ajouter ou basculer entre plusieurs wallets

Yoroi peut gérer plusieurs portefeuilles.

Si vous êtes dans une vue portefeuille, vous verrez une liste déroulante dans le coin supérieur droit, qui devrait énumérer vos portefeuilles existants et vous permettre d'ajouter un nouveau wallet (`Add new wallet`).

![Add Another Wallet from Wallet screen](https://raw.githubusercontent.com/cardano-community/support-faq/images/docs/images/yoroi-multiple-wallet-add.jpg ':size=30%')

Si vous êtes à l'écran `My Wallets`, vous avez la possibilité d'ajouter un nouveau wallet en haut.

![Add Another Wallet from My Wallets screen](https://raw.githubusercontent.com/cardano-community/support-faq/images/docs/images/yoroi-multiple-wallet-add2.jpg ':size=30%')

Dans les deux cas, vous accédez à l'écran ci-dessous :

![Add new wallet](https://raw.githubusercontent.com/cardano-community/support-faq/images/docs/images/yoroi-multiple-add-new-wallet.jpg ':size=30%')

Vous pouvez poursuivre les étapes suivantes en suivant les instructions [créer un wallet](#create-a-walt-on-yoroi-extension-or-addon) ci-dessus.
